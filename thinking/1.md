# I Ching AI 页面设计分析

## 1. 颜色系统

```css
:root {
  --primary: #8a2be2; /* 主色：紫色 */
  --secondary: #ffd700; /* 辅助色：金色 */
  --dark: #333; /* 深色文本 */
  --light: #f8f8f8; /* 浅色背景 */
  --accent: #ff6b6b; /* 强调色 */
  --text: #333; /* 文本颜色 */
  --bg: #fff; /* 背景色 */
}
```

## 2. 字体与排版

- 字体系列: `'Segoe UI', Tahoma, Geneva, Verdana, sans-serif`
- 行高: 1.6
- 标题大小: 48px (h1), 逐级递减
- 正文文本: 16-20px
- 小文本: 14px 或更小

## 3. 布局结构

- 容器最大宽度: 1200px
- 内边距: 20px
- 卡片最大宽度: 650px

### 4.2 步骤指示器

- 步骤指示器使用圆形数字标记
- 活动步骤使用紫色(#8A2BE2)背景和白色文字
- 非活动步骤使用灰色背景(#eee)和灰色文字(#777)
- 步骤之间有连接线
- 圆形尺寸: 40px x 40px

```css
.step-number {
  width: 40px;
  height: 40px;
  border-radius: 50%;
  background-color: #eee;
  color: #777;
  display: flex;
  align-items: center;
  justify-content: center;
  font-weight: bold;
  margin-bottom: 8px;
}

.process-step.active .step-number {
  background-color: var(--primary); /* #8A2BE2 */
  color: white;
}
```

## 6. 响应式设计要点

- 移动设备上减小内边距
- 调整字体大小
- 简化布局

## 7. 实现注意事项

1. 确保使用正确的颜色值 (#8A2BE2 而非通用的 purple-600)
2. 保持一致的圆角半径 (按钮、输入框等)
3. 保持一致的阴影效果
4. 正确实现步骤指示器的连接线和状态
5. 确保文本大小和间距与原始设计匹配
6. 使用正确的字体系列
7. 实现所有交互动画效果

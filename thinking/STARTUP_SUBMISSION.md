## 🏗️ 技术架构

### 前端技术栈

- **框架**: Next.js 14 (App Router)
- **UI 库**: React + Tailwind CSS + Shadcn/ui
- **状态管理**: React Context + Custom Hooks
- **国际化**: next-intl (6 语言支持)
- **主题系统**: next-themes (深色/浅色模式)
- **动画**: Framer Motion
- **图标**: Lucide React, React Icons

### 后端技术栈

- **API**: Next.js API Routes
- **认证**: NextAuth.js 5.0 (Google, GitHub, Email)
- **数据库**: Supabase (PostgreSQL)
- **支付系统**: Stripe 集成
- **AI 集成**: Anthropic Claude 4, OpenAI, DeepSeek
- **视频生成**: FAL.AI (12+模型集成)
- **文件存储**: AWS S3

### AI 模型集成

- **对话模型**: Claude 4 Sonnet, GPT-4, DeepSeek
- **视频生成模型**:
  - Kling 1.6 & 2.0 (快手)
  - MiniMax (文本/图片转视频)
  - Luma Dream Machine
  - Google Veo 2
  - Pixverse
  - Hunyuan (腾讯)
  - Mochi (开源)
  - Haiper 2.0

# Veo3 AI - Advanced AI Video Generation Platform

## Project Overview

Veo3 AI (veo3ai.io) is a comprehensive AI-powered video generation platform that enables users to create professional-quality videos from text prompts or images. The platform offers multiple AI video generation providers and a complete credit-based subscription system with global payment processing.

## Core Features & Capabilities

### Video Generation
- **Text-to-Video**: Generate videos from detailed text descriptions with cinematic quality
- **Image-to-Video**: Animate static images into dynamic video content
- **AI Video Effects**: Specialized effects including muscle transformation, kissing videos, 360° rotation
- **Multiple AI Providers**: Veo3, Seedance, and other leading AI video models
- **High-Quality Output**: Up to 1080p resolution with professional motion physics
- **Customizable Settings**: Aspect ratios (16:9, 9:16, 1:1), durations (5-10 seconds), various resolutions

### User Experience
- **Intuitive Interface**: Clean, modern design with easy-to-use video generation tools
- **Real-time Progress**: Live status tracking for video generation with estimated completion times
- **Video History**: Complete generation history with playback, download, and regeneration options
- **Multi-language Support**: English and Russian localization
- **Responsive Design**: Optimized for desktop and mobile devices

### Credit & Payment System
- **Credit-based Pricing**: 0.25 USD = 10 credits, with dynamic pricing based on model and settings
- **Multiple Payment Providers**: Creem (primary), Stripe (international), Payssion V2 (Russian market)
- **Subscription Plans**: Monthly and yearly memberships with automatic credit distribution
- **Free Trial**: New users receive free credits to explore the platform

### Authentication & User Management
- **Multiple Sign-in Options**: Google, GitHub, Apple, Email authentication
- **User Dashboard**: Credits management, subscription details, order history
- **Invite System**: Referral program with bonus credits for successful invites
- **API Access**: API keys for developers and integrations

## Technical Architecture

### Frontend Technology
- **Next.js 14**: Modern React framework with App Router and Server Components
- **TypeScript**: Full type safety throughout the application
- **Tailwind CSS**: Utility-first CSS framework with custom animations
- **Shadcn/ui & Radix**: High-quality UI components and primitives
- **Framer Motion**: Advanced animations and micro-interactions

### Backend & Infrastructure
- **Supabase**: PostgreSQL database with Row Level Security
- **NextAuth.js 5.0**: Secure authentication with multiple providers
- **Cloudflare Pages**: Global deployment with edge optimization
- **Webhooks**: Real-time status updates from video generation providers

### AI Integration
- **Kie.ai (Veo3)**: Premium video generation with 3 credits/second, 8-second max
- **Volcano Engine (Seedance)**: High-quality video generation with 2 credits/second, 10-second max
- **Provider Factory Pattern**: Unified interface for multiple AI video providers
- **Intelligent Routing**: Automatic provider selection based on requirements

### Security & Compliance
- **Row Level Security**: Database-level access control
- **Webhook Verification**: HMAC-SHA256 signature validation for payments
- **Input Validation**: Zod schemas for all API requests
- **Content Filtering**: Automatic sensitive data filtering in logs

## User Journey & Workflows

### Video Creation Process
1. **Authentication**: Sign in with preferred provider (Google, GitHub, Apple, Email)
2. **Generation Type Selection**: Choose text-to-video, image-to-video, or video effects
3. **Content Input**: Provide text prompt (30+ characters) or upload image (JPG/PNG, max 10MB)
4. **Configuration**: Select model, aspect ratio, duration, and quality settings
5. **Credit Calculation**: System calculates required credits based on settings
6. **Generation**: AI processes request with real-time status updates
7. **Result**: Download, view, or regenerate completed videos

### Subscription & Payment Flow
1. **Plan Selection**: Choose from available subscription tiers
2. **Payment Processing**: Intelligent routing to appropriate payment provider based on location
3. **Credit Distribution**: Automatic credit allocation upon successful payment
4. **Subscription Management**: Cancel, modify, or upgrade plans through user dashboard

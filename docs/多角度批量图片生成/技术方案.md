# 多角度批量图片生成技术方案

> **开发状态**: ✅ 已完成 (2025-12-02)
>
> **构建验证**: `pnpm build` 通过

## 需求概述

为广告电商和漫剧创作者提供多角度批量图片生成功能。用户输入提示词 + 参考图后，系统通过 LLM 自动扩展提示词，一次生成 6-12 张覆盖多视角/多情境的图片。

## 核心设计决策

| 决策项 | 方案 |
|--------|------|
| 数据存储 | 一条记录，`image_urls` 数组存多张 |
| 积分计费 | N 张 × 单价 (2 credits/张) |
| Agent 标签 | 橙色/金色 (amber-500) |
| 提示扩展 | LLM (OpenAI 兼容 API，默认 Evolink) |
| 扩展提示可见性 | 默认隐藏 |
| 并发控制 | p-limit，最大 6 并发 |

## 二、后端实现

### 2.1 提示词扩展服务

**文件**: `services/promptExpander.ts`

```typescript
import OpenAI from 'openai';

// 使用 OpenAI 兼容 API (支持 Evolink/OpenAI/Claude 等)
const llmClient = new OpenAI({
  apiKey: process.env.EVOLINK_API_KEY,
  baseURL: process.env.EVOLINK_BASE_URL || 'https://api.evolink.ai/v1',
});

export type UserContext = 'ecommerce' | 'comic' | 'general';

export async function expandImagePrompts(
  originalPrompt: string,
  imageCount: number,
  hasReferenceImages: boolean = false,
  userContext: UserContext = 'general'
): Promise<string[]> {
  // 构建系统提示词
  const systemPrompt = buildSystemPrompt(userContext);
  const userMessage = buildUserMessage(originalPrompt, imageCount, hasReferenceImages);

  const response = await llmClient.chat.completions.create({
    model: process.env.EVOLINK_MODEL || 'gemini-3-pro-preview',
    messages: [
      { role: 'system', content: systemPrompt },
      { role: 'user', content: userMessage },
    ],
    response_format: { type: 'json_object' },
  });

  return parseExpandedPrompts(response.choices[0].message.content, imageCount, originalPrompt);
}
```

### 2.2 Agent 图片生成服务

**文件**: `services/agentImageService.ts`

```typescript
import pLimit from 'p-limit';
import { expandImagePrompts, UserContext } from './promptExpander';
import { aiServiceManager } from './AIServiceManager';
import { createImageGeneration } from '@/models/imageGeneration';
import { decreaseCreditsMultiPool } from '@/services/credit';
import { calculateImageCredits } from '@/config/image-models';

const MAX_CONCURRENT = 6;

export async function generateAgentImages(
  params: AgentGenerationParams
): Promise<AgentGenerationResult> {
  // 1. 扩展提示词
  const expandedPrompts = await expandImagePrompts(
    prompt, imageCount, hasReferenceImages, userContext
  );

  // 2. 计算并扣除积分
  const singleImageCredits = calculateImageCredits(model, resolution);
  const totalCredits = singleImageCredits * imageCount;
  const creditDeduction = await decreaseCreditsMultiPool({...});

  // 3. 创建数据库记录
  const imageGeneration = await createImageGeneration({
    is_agent_mode: true,
    agent_image_count: imageCount,
    expanded_prompts: expandedPrompts,
    // ...
  });

  // 4. 并发生成图片 (最大 4 并发)
  const limit = pLimit(MAX_CONCURRENT);
  const tasks = expandedPrompts.map((expandedPrompt, idx) =>
    limit(async () => {
      const result = await aiServiceManager.generateImage(provider, {...});
      return { index: idx, success: true, taskId: result.taskId };
    })
  );

  const results = await Promise.all(tasks);

  return {
    generationId: imageGeneration.id,
    status: 'processing',
    imageCount,
    tasks: results,
  };
}
```

## 七、部分失败退款逻辑

采用**按比例退款**策略：成功 N 张扣 N 张积分，失败的自动退款。

```typescript
// 在回调处理中
async function handleAgentPartialRefund(
  imageGeneration: ImageGeneration,
  failedCount: number
): Promise<void> {
  const singleImageCredits = calculateImageCredits(
    imageGeneration.model_id,
    imageGeneration.metadata?.resolution
  );
  const refundAmount = singleImageCredits * failedCount;

  // 按原扣费池逆序退款
  const poolsReversed = [...creditDeduction.pools].reverse();
  for (const pool of poolsReversed) {
    const refundFromPool = Math.min(pool.deducted, remainingRefund);
    await increaseCredits({
      user_uuid: imageGeneration.user_id,
      trans_type: CreditsTransType.RefundImageGenerationFailed,
      credits: refundFromPool,
      order_no: pool.order_no,
    });
  }
}
```

## 八、风险和注意事项

1. **API 限流**: 使用 p-limit 控制最大并发数为 6，避免触发 Nano Banana API 速率限制
2. **超时处理**: LLM 提示扩展设置 30s 超时；批量图片生成总超时 10 分钟
3. **进度反馈**: 前端展示 "Generating: 3/8 images" 形式的实时进度
4. **R2 存储**: 批量图片逐张上传到 R2
5. **环境变量**: 确保配置 `EVOLINK_API_KEY` 否则提示扩展功能将使用 fallback 逻辑

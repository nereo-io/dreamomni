# 多角度批量图片生成需求草案

## 目标与背景

- 针对广告电商与漫剧创作者，用户输入提示词与参考图后，一次生成 6~12 张图片，覆盖多视角/多情境，减少反复提炼提示词的成本。
- 系统后台自动扩展用户提示词，补充角度、场景、光效、辅助物品/角色姿态，力求超出用户预期的创意与覆盖度；默认不暴露扩展提示，仅在高级模式可查看/编辑。

## 核心用户场景（Use Cases）

- **广告电商**：单个商品电商图，自动生成多角度（正/侧/俯）、多场景（棚拍、户外、生活化）、多道具（配饰、包装、陈列）。可选择风格模板（极简、科技、复古、奢侈品）。
- **电商素材批量迭代**：给定主图 + 品牌调性，生成一组可直接用于 A/B 测试的海报/场景图，自动排列文案/徽标留白。
- **漫剧关键帧**：输入故事梗概 + 角色设定（含参考图），自动展开多个情节点的关键帧（开场/冲突/转折/高潮/结尾），保证角色一致性与分镜多样。

## 关键体验

- 单次输入 → 自动多角度扩展 → 一次输出 6~12 张；无需用户自己拆提示。
- 对电商：角度/场景/光线/道具自动覆盖，便于 A/B。
- 对漫剧：故事节点自动拆分关键帧，角色保持一致。

## 功能需求（结合现有项目能力）

- **输入**：提示词必填；参考图 1~3 张可选；参数：、纵横比或分辨率（沿用现有 `aspect_ratio` / `image_size`）、模型。- 增加一个 agent 的开关，用户打开之后可以选择要生成的图片数量。数量可选范围，6~12 张
- **提示扩展**：后台生成 6 ～ 12 条扩展提示，覆盖角度/场景/光效/道具或剧情节点，不展示到前端
- **生成模式**：文本生图 + 参考图生图（保持主体/角色一致）；支持轻度重绘（背景替换、道具微调）。
- **批量输出与标注**：一次返回 6~12 张

-可以继续使用现在的图片历史的展示的框架。现在展示的框架是一次，请展示一个图片，那可以变成一次展示 6~12 个图片。同时在用户提示词下方的标签位置展示 agent 的标签表现，可以用一种突出点颜色放在首位展示

## 📊 数据库结构详解

### 主要表结构

#### **1. image_generations 表**

存储所有图片生成任务的核心信息：

```sql
-- 核心字段
id                    UUID      # 主键
user_id              TEXT      # 用户UUID (关联users表)
task_id              TEXT      # 通用任务ID
nano_banana_task_id  TEXT      # NanoBanana专用任务ID
model_id             TEXT      # AI模型标识
prompt               TEXT      # 提示词
mode                 ENUM      # 生成模式 (text-to-image, image-edit等)
source               ENUM      # 创建来源 (web, api, mobile等)
status               ENUM      # 状态 (PENDING, COMPLETED, FAILED等)
credits_used         INTEGER   # 消耗积分
image_urls           JSONB     # 生成的图片URL数组
created_at           TIMESTAMPTZ
updated_at           TIMESTAMPTZ
completed_at         TIMESTAMPTZ
```

#### **2. image_generation_credits 表**

详细记录积分消耗情况：

```sql
-- 核心字段
id                    UUID      # 主键
user_uuid            TEXT      # 用户UUID
image_generation_id  UUID      # 关联image_generations表
credits_consumed     INTEGER   # 消耗积分数
credits_before       INTEGER   # 消耗前余额
credits_after        INTEGER   # 消耗后余额
model_id             TEXT      # 使用的模型
base_cost            INTEGER   # 基础成本
final_cost           INTEGER   # 最终成本
promotion_code       TEXT      # 促销代码
created_at           TIMESTAMPTZ
```

### 枚举类型

```sql
-- 生成状态
image_generation_status: PENDING, IN_QUEUE, IN_PROGRESS, COMPLETED, FAILED, SAVED_TO_R2

-- 生成来源  
image_generation_source: web, api, mobile, extension

-- 生成模式
image_generation_mode: text-to-image, image-edit, image-to-image, inpainting, outpainting
```

## 🔧 在代码中使用

### 1. **导入模型函数**

```typescript
import {
  createImageGeneration,
  updateImageGenerationByNanoBananaTaskId,
  getUserImageGenerations,
  getUserImageGenerationStats
} from '@/models/imageGeneration';

import type {
  ImageGeneration,
  CreateImageGenerationParams,
  UpdateImageGenerationParams,
  ImageGenerationHistoryItem
} from '@/types/image.d';
```

### 2. **创建图片生成记录**

```typescript
// 创建新的图片生成任务
const newGeneration = await createImageGeneration({
  user_id: 'user-uuid-here',
  model_id: 'nano-banana',
  prompt: 'A beautiful sunset over mountains',
  mode: 'text-to-image',
  source: 'web',
  nano_banana_task_id: 'task-123',
  credits_used: 1,
  status: 'PENDING'
});
```

### 3. **更新生成状态**

```typescript
// 当收到 NanoBanana 回调时更新状态
await updateImageGenerationByNanoBananaTaskId('task-123', {
  status: 'COMPLETED',
  image_urls: ['https://cdn.example.com/generated-image.jpg'],
  completed_at: new Date().toISOString()
});
```

### 4. **查询用户历史**

```typescript
// 获取用户的图片生成历史
const { data: history, total } = await getUserImageGenerations(
  'user-uuid-here',
  20,  // limit
  0    // offset
);

// 获取用户统计信息
const stats = await getUserImageGenerationStats('user-uuid-here');
```

## 📈 统计和分析

### 预定义统计视图

```sql
-- 用户统计
SELECT * FROM user_image_generation_stats WHERE user_id = 'user-uuid';

-- 模型使用统计
SELECT * FROM model_usage_stats ORDER BY total_usage DESC;

-- 每日生成统计
SELECT * FROM daily_generation_stats 
WHERE generation_date >= CURRENT_DATE - INTERVAL '30 days';
```

### 自定义查询示例

```sql
-- 查找最受欢迎的提示词
SELECT 
  prompt,
  COUNT(*) as usage_count,
  AVG(credits_used) as avg_credits
FROM image_generations 
WHERE status = 'COMPLETED'
  AND created_at >= NOW() - INTERVAL '7 days'
GROUP BY prompt 
ORDER BY usage_count DESC 
LIMIT 10;

-- 模型成功率分析
SELECT 
  model_id,
  COUNT(*) as total_attempts,
  COUNT(*) FILTER (WHERE status = 'COMPLETED') as successful,
  ROUND(
    COUNT(*) FILTER (WHERE status = 'COMPLETED') * 100.0 / COUNT(*), 
    2
  ) as success_rate
FROM image_generations 
GROUP BY model_id 
ORDER BY success_rate DESC;

-- 积分消耗趋势
SELECT 
  DATE_TRUNC('day', created_at) as date,
  SUM(credits_used) as daily_credits,
  COUNT(*) as daily_generations,
  COUNT(DISTINCT user_id) as active_users
FROM image_generations 
WHERE created_at >= NOW() - INTERVAL '30 days'
GROUP BY DATE_TRUNC('day', created_at)
ORDER BY date DESC;
```

## 🔒 权限和安全

### 行级安全 (RLS)

数据库已启用行级安全，确保：

- ✅ 用户只能查看自己的生成记录
- ✅ 用户只能修改自己的记录
- ✅ 防止跨用户数据泄露
- ✅ 管理员权限预留接口

### 数据完整性

- ✅ 外键约束确保数据一致性
- ✅ 检查约束防止无效数据
- ✅ 触发器自动维护计算字段
- ✅ 唯一约束防止重复记录

## 🚨 故障排除

### 常见问题

#### 1. **迁移失败**

```sql
-- 检查是否有依赖冲突
SELECT conname, conrelid::regclass, confrelid::regclass 
FROM pg_constraint 
WHERE contype = 'f' 
  AND (conrelid::regclass::text LIKE '%image_generation%' 
       OR confrelid::regclass::text LIKE '%image_generation%');

-- 检查枚举类型是否已存在
SELECT typname FROM pg_type WHERE typname LIKE '%image_generation%';
```

#### 2. **性能问题**

```sql
-- 检查索引是否被使用
SELECT 
  indexrelname,
  idx_scan,
  idx_tup_read,
  idx_tup_fetch
FROM pg_stat_user_indexes 
WHERE relname = 'image_generations'
ORDER BY idx_scan DESC;

-- 检查是否需要重建索引
SELECT 
  schemaname,
  tablename,
  attname,
  correlation
FROM pg_stats 
WHERE tablename = 'image_generations' 
  AND correlation < 0.1;
```

#### 3. **数据不一致**

```sql
-- 检查外键完整性
SELECT ig.id, ig.user_id 
FROM image_generations ig 
LEFT JOIN users u ON ig.user_id = u.uuid 
WHERE u.uuid IS NULL;

-- 检查图片数量计算
SELECT id, image_urls, image_count 
FROM image_generations 
WHERE image_urls IS NOT NULL 
  AND image_count != jsonb_array_length(image_urls);
```

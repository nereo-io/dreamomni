# 图片生成功能架构升级

## 🎯 升级目标

将图片生成界面重构为更灵活的功能分类架构，支持 "Text to Image" 和 "Image to Image" 两大类功能，每个分类下可扩展多种AI模型。

## ✨ 主要改进

### 1. **功能分类清晰**
- 🎨 **Text to Image**: 从文本描述生成图片
- 🖼️ **Image to Image**: 编辑和转换现有图片
- 🔄 **智能切换**: 切换分类时自动重置相关状态

### 2. **模型扩展架构**
- 📦 **当前可用**: Nano Banana 系列模型
- 🚀 **占位模型**: Stable Diffusion、Midjourney、ControlNet 等
- 🏷️ **状态标识**: "Coming Soon" 标签区分可用和占位模型

### 3. **直观的UI设计**
- 🎯 **模型卡片**: 每个模型独立卡片显示详细信息
- 💰 **积分显示**: 清晰的积分消耗标识
- 🔵 **选中状态**: 蓝色边框和背景突出当前选择

## 🔄 交互流程

### 功能切换流程
1. **选择功能类型** → Text to Image / Image to Image
2. **自动重置状态** → 清空图片上传（如果切换到Text to Image）
3. **自动选择模型** → 选择该类型下第一个可用模型
4. **更新界面** → 更新提示词标签和占位符

### 模型选择流程
1. **查看可用模型** → 显示当前功能类型下的所有模型
2. **区分状态** → 可用模型可点击，占位模型显示"Coming Soon"
3. **选择模型** → 点击可用模型进行选择
4. **更新积分** → 显示新选择模型的积分消耗

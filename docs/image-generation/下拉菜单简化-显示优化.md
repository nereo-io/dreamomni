# 下拉菜单简化 - 显示优化

## 🎯 优化目标

简化下拉菜单选中后的显示内容，只保留模型名称和积分信息，提升界面简洁性和用户体验。

## 📋 优化对比

### 优化前 (默认显示)
```
AI Model
┌─────────────────────────────────────────┐
│ Nano Banana                       ▼   │  ← 使用默认的SelectValue
└─────────────────────────────────────────┘
```

### 优化后 (自定义显示)
```
AI Model
┌─────────────────────────────────────────┐
│ Nano Banana                    [1 ⚡] ▼│  ← 只显示名称和积分
└─────────────────────────────────────────┘
```

## ✨ 设计亮点

### 1. **简洁的选中显示**
- 🎯 **核心信息**: 只显示模型名称和积分消耗
- 📏 **空间高效**: 去除冗余的描述信息
- 👁️ **清晰易读**: 重要信息一目了然

### 2. **保留完整选项**
- 📋 **展开详情**: 点击展开时仍显示完整信息
- 🏷️ **状态标识**: "Coming Soon" 标签保持可见
- 📝 **详细描述**: 提供商和描述信息在选择时可见

### 3. **一致的积分显示**
- 💰 **积分徽章**: 琥珀色积分图标和数字
- 🎨 **视觉统一**: 与选项中的积分显示保持一致
- ⚡ **即时可见**: 用户无需展开即可看到消费

## 🔧 技术实现

### 自定义SelectValue
```typescript
<SelectValue>
  {currentModel ? (
    <div className="flex items-center justify-between w-full">
      <span className="font-medium">{currentModel.name}</span>
      <div className="flex items-center gap-1 text-amber-400">
        <Coins className="h-3 w-3" />
        <span className="text-xs font-medium">{currentModel.credits}</span>
      </div>
    </div>
  ) : (
    "Select a model"
  )}
</SelectValue>
```

### 显示逻辑
- ✅ **有选择**: 显示模型名称 + 积分徽章
- ❌ **无选择**: 显示占位符文字
- 🔄 **动态更新**: 根据currentModel状态自动更新

| **信息密度** | 低 | 高 | ✅ 提升效率 |
| **用户体验** | 一般 | 优秀 | ✅ 更直观 |

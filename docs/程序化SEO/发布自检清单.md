# 发布自检清单（上线前必查）

适用范围：Comparison / Alternatives / Best‑of / How‑To 四类页面。

## 必备要素
- 字数 ≥ 900；首屏摘要 80–120 字；结构清晰（H2/H3 与模板一致）
- 素材 ≥ 2 张图或 1 段视频；均有 alt 文案与来源授权
- FAQ ≥ 3 条；包含用户最关心的权限/配额/价格/地区/水印等
- 来源与“最近更新”日期明确（YYYY‑MM‑DD）

## SEO 与元信息
- Title ≤ 60 字符，唯一；Description 150–160 字，含主关键词但不过度重复
- Slug 与主关键词一致；URL 规则按栏目规范（compare/tools/blog）
- Canonical 唯一；如与其它页高度相似，确定主/从页避免内耗
- 内链 ≥ 3：模型页/集合页/对比或教程；外链仅指向权威来源

## 质量与合规
- 不虚构数据；价格/参数/功能对齐来源页，标注时间戳
- 明确推广/合作标识（如有），区分编辑与广告内容
- 图片/视频版权可用；第三方引用遵循许可

## 触发 noindex 的情形（先发草稿，后优化）
- 信息量不足（<900 字）或缺核心素材；与既有页面高度重复且暂无合并方案
- 主题与意图冲突（两个页面争夺同一核心关键词）

## 发布后两周复盘（最低门槛）
- 收录：URL Inspection 收录成功
- 曝光：Impressions 明显上升（对比前一周期）
- CTR：≥ 1.5%（新域名可放宽）；未达标则改标题/摘要与首屏
- 行为：停留与滚动率达标；弱页合并或继续 noindex

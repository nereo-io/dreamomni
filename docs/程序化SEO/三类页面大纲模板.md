# 三类页面大纲模板（可复制使用）

本文件提供可直接套用的大纲。占位变量使用大括号表示，如 {ModelA}。

## 1) Comparison 对比页（内容页 / /compare/{a}-vs-{b}）
- 标题：{A} vs {B}：画质/音频/一致性/速度/价格对比 | Veo3 AI
- 首屏摘要（80–120字）：一句话结论 + 适用人群
- H2 结论：谁更适合你？
  - 何时选 {A}：{3–5 条场景}
  - 何时选 {B}：{3–5 条场景}
- H2 差异点矩阵
  - 画质（分辨率/细节/风格）：{A} vs {B}
  - 音频能力（原生/质量/同步）：{A} vs {B}
  - 角色一致性/镜头控制：{A} vs {B}
  - 速度/稳定性/成功率：{A} vs {B}
  - 价格/额度/速率限制：对照表
  - 商用/水印/地区可用性：对照表
- H2 实操建议（1–2 个工作流）
  - 工作流 1（3 步）
  - 工作流 2（3 步）
- H2 常见问题（FAQ，≥3）
- 相关链接：{模型 A}、{模型 B}、相关 How‑To、集合/替代页
- 来源与更新时间：列出引用链接与日期

## 2) Alternatives/Best‑of 工具集合页（/tools/{topic}）
- 标题：{Topic}：Top {N} 清单 | Veo3 AI
- 首屏摘要：适用场景 + 选择要点（价格/音频/4K/速度/开源等）
- H2 如何选择（评估维度）
  - 维度 1：{比如 画质与分辨率}
  - 维度 2：{比如 音频/一致性}
  - 维度 3：{比如 速度/价格/配额}
- H2 Top {N}
  - 条目结构：名称 + 一句话定位 + 关键特性 + 优缺点 + 适用人群 + 价格区间 + 示例图/视频
- H2 快速上手（3 步）
- H2 常见问题（FAQ，≥3）
- 相关链接：对应模型页、对比页、How‑To
- 来源与更新时间

## 3) How‑To 内容页（/blog/how-to-{task}）
- 标题：How to {Task} with {Model/Tool}
- 首屏摘要：一句话方案 + 前置条件/素材要求
- H2 步骤总览（3–5 步）
  - 步骤 1：标题 + 要点 + 常见错误
  - 步骤 2：标题 + 要点 + 常见错误
  - 步骤 3：标题 + 要点 + 常见错误
- H2 结果优化（可选 2–3 条）
- H2 常见问题（FAQ，≥3）
- 相关链接：关联模型/工具、集合页、对比页
- 来源与更新时间

## LLM 首稿提示语（可直接复制）
- Comparison：写 {A} vs {B} 对比文，必须含“结论/差异点矩阵/何时选A或B/FAQ(3条)”，≤1200字，口语化但给依据，不可虚构数据。
- Alternatives：写 “{X} alternatives” 榜单，列 Top{N}（简介/优缺点/适用人群/价格区间/一行理由），附“如何选择”和 FAQ(3条)，≤1400字。
- How‑To：写“如何用 {模型/工具} 完成 {任务}”，3–5 步，每步含注意事项与常见错误，结尾给相关模型与集合页，≤1000字。

# 任务：调研 Veo3 Reference-to-Video 功能特点

## 背景

Veo3 AI 平台即将推出"参考生视频"(Reference-to-Video)功能。用户可以上传 1-3 张参考图，生成保持角色一致性的视频。

## 你的任务

通过网络搜索，全面了解 Veo3 的 Reference-to-Video 功能，并从**用户视角**整理调研结果。

### 调研重点

1. **核心功能**
   - Reference-to-Video 到底是什么？
   - 与普通 Image-to-Video 的本质区别是什么？
   - 技术实现原理（简化版，用户能理解的）

2. **核心卖点**
   - 用户为什么要用这个功能？（真实痛点）
   - 相比竞品的独特优势是什么？
   - 哪些使用场景最有价值？

3. **使用体验**
   - 用户需要准备什么样的参考图？
   - 操作流程是什么？（具体步骤）
   - 生成效果如何？有什么限制？

4. **常见问题**
   - 用户最关心的 6-8 个问题
   - 每个问题的准确答案（不要编造）

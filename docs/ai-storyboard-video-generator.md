# AI 辅助分镜与视频生成工具需求文档

## 1. 背景与价值
- 面向创意工作者（视频制作者、电商广告创业者）的一站式分镜与视频草稿生成工具。
- 减少创意发散与分镜反复修改的人力成本，缩短从想法到视频草稿的周期。
- 若团队已有成熟分镜产线，增益有限；对中小团队/个人制作者提效显著。

## 2. 目标
- 让用户基于提示词与参考图快速生成多套分镜关键帧，选中后自动生成短视频段落并拼接成完整草稿。
- 保持镜头风格、角色、光线的一致性，支持快速迭代与重试。

## 3. 用户与 Use Case
- 用户：创意工作者、独立视频创作者、电商广告创业者。
- 典型任务：围绕一个产品或故事线，生成 6–12 张关键帧 → 生成对应短视频 → 拼接成可预览/下载的草稿视频。
- 痛点：构思耗时、分镜修改反复、初版视频制作成本高。

## 4. 用户流程
1) 输入与理解：文字提示词 + 参考图（可多张）；可选参数（风格、时长、镜头数）。
2) 分镜生成：输出多张关键帧，附镜头描述、时长建议、运动与转场提示；支持收藏、删除、重生成。
3) 视频生成：对选中关键帧执行首帧生成视频或参考图生成视频（保持构图、角色、风格）。
4) 拼接与导出：挑选镜头视频，拖拽排序、微调时长、套用基础转场模板；生成并预览完整视频草稿，支持下载。

## 5. 核心功能需求
- 输入：文本提示、多图上传；场景、风格、时长参数化；任务创建与排队提示。
- 分镜：批量关键帧生成（至少 6–12 张）；镜头标签（景别、角度、光线、情绪）；一致性控制（角色、风格锁定）。
- 视频：两种生成模式（首帧、参考）；每段可重试；生成状态与失败原因提示。
- 编辑：时间线拼接、顺序拖拽、时长微调、基础转场；一键导出草稿。
- 账号与存储：历史项目与任务列表；素材与生成结果归档。

## 6. 非功能需求
- 性能：分镜生成 ≤ 15s；单段视频生成 ≤ 60s（受队列与模型影响可调）；进度与排队状态可见。
- 稳定性：失败自动重试 1–2 次；用户可取消任务。
- 扩展性：模型/推理后端可替换；支持队列与并发控制。
- 监控：生成成功率、耗时、留存/转化埋点；告警与日志。
- 安全与合规：敏感或违规内容拦截；参考素材版权提示与来源标注。

## 7. MVP 范围
- 必须：文本+参考图输入；分镜关键帧生成与筛选；首帧或参考生成视频；时间线拼接与草稿导出；进度与错误反馈。
- 可延后：高级转场模板；多人协作；品牌或角色长程记忆；自动配乐、调色；团队权限与配额。

## 8. 成功度量（示例）
- 生成成功率 ≥ 95%；分镜首屏等待 ≤ 15s。
- 单项目从输入到草稿导出的中位时长 ≤ 5 分钟。
- 用户二次使用率、镜头选择转化率、导出率提升。

## 路由逻辑规则

### 首页访问逻辑（/）

1. **未登录用户**: 访问 `/` → 显示 landing page
2. **已登录用户**: 访问 `/` → 自动重定向到 `/home`

### 应用页面访问逻辑（/home 及其子路由）

1. **未登录用户**: 访问 `/home` → 允许访问，显示应用界面（但功能受限，引导登录）
2. **已登录用户**: 访问 `/home` → 正常显示应用界面，功能完整

### 登录后重定向

- 用户登录成功后，自动跳转到 `/home` 页面

### 实现状态

- ✅ 已实现首页登录状态检测和重定向
- ✅ 已实现登录后跳转到 `/home`
- ✅ `/home` 路由对所有用户开放

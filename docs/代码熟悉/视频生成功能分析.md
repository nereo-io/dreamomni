# Veo3 AI 视频生成功能深度分析

## 一、功能概述

Veo3 AI 平台提供了两种核心的视频生成模式：
1. **文字生成视频 (Text-to-Video)** - 通过文本描述生成视频内容
2. **图片生成视频 (Image-to-Video)** - 将静态图片转换为动态视频

这两种模式共享底层的视频生成架构，但在输入处理、模型选择和参数配置上有所差异。

export class ProviderFactory {
  static getProvider(modelId: string): VideoProvider {
    const modelConfig = getVideoModel(modelId);
    
    switch (modelConfig.provider) {

        return new VolcanoProvider(process.env.ARK_API_KEY);

  }
}

## 六、状态管理与轮询机制

### 6.1 状态流转

```
PENDING → PROMPT_OPTIMIZING → IN_QUEUE → IN_PROGRESS → COMPLETED → SAVED_TO_R2
                                               ↓
                                            FAILED
```

### 6.2 轮询实现

```typescript
// hooks/useVideoGeneration.ts
const pollStatus = useCallback(async (id: string) => {
  const maxAttempts = 100;
  const interval = 3000; // 3秒轮询一次
  
  const poll = async () => {
    if (attempts >= maxAttempts) {
      toast.error("查询超时，请手动刷新");
      return;
    }
    
    const status = await checkStatus(id);
    
    if (status.status === "COMPLETED" || status.status === "FAILED") {
      // 停止轮询
      if (status.status === "COMPLETED") {
        toast.success("视频生成完成！");
      }
      return;
    }
    
    // 继续轮询
    setTimeout(poll, interval);
  };
  
  poll();
}, []);
```

## 七、积分计算系统

### 7.1 计算规则

```typescript
// config/video-models.ts
export function calculateCredits(
  modelId: string,
  durationSeconds: number,
  hasAudio: boolean = false,
  resolution: string = "1080p"
): number {
  const model = getVideoModel(modelId);
  
  // 基础积分 = 每秒积分 × 时长
  let credits = model.perSecondCredits * durationSeconds;
  
  // 音频附加费用
  if (hasAudio && model.supportsAudio) {
    credits += model.audioPremiumCredits || 0;
  }
  
  // 高分辨率附加费用（未来扩展）
  if (resolution === "4K") {
    credits *= 1.5;
  }
  
  return Math.ceil(credits);
}
```

## 九、错误处理与容错

### 9.1 前端错误处理

```typescript
// 1. 网络错误重试
const retryRequest = async (fn: Function, retries = 3) => {
  for (let i = 0; i < retries; i++) {
    try {
      return await fn();
    } catch (error) {
      if (i === retries - 1) throw error;
      await sleep(1000 * Math.pow(2, i)); // 指数退避
    }
  }
};

// 2. 用户友好的错误提示
const handleError = (error: any) => {
  if (error.code === 'INSUFFICIENT_CREDITS') {
    toast.error("积分不足，请充值后再试");
    setShowPricingModal(true);
  } else if (error.code === 'RATE_LIMITED') {
    toast.error("请求过于频繁，请稍后再试");
  } else {
    toast.error(error.message || "生成失败，请重试");
  }
};
```

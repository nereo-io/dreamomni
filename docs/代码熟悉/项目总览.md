# Veo3 AI 项目总览

## 一、项目概述

**Veo3 AI** (veo3ai.io) 是一个专注于提供卓越视频生成体验的 AI 平台。该平台集成了多种先进的视频生成模型，提供文本生成视频、图片生成视频等功能，并配备了完整的用户管理、支付、订阅和积分系统。

### 核心特点
- 多模型视频生成支持（Volcano Engine、APICore Veo3、fal.ai、KieAi等）
- Claude 4 Sonnet 集成的智能聊天功能
- 多语言国际化支持（基于 next-intl）
- 完整的用户管理和认证系统
- 多支付渠道集成（Creem、Stripe、Payssion）
- 积分和订阅管理系统
- 响应式设计，支持移动端

## 二、技术架构

### 2.1 技术栈

#### 前端技术
- **框架**: Next.js 14 with App Router
- **语言**: TypeScript (严格模式)
- **样式**: Tailwind CSS + CSS-in-JS
- **UI组件库**: 
  - Shadcn/ui (基于 Radix UI primitives)
  - 自定义组件库
- **状态管理**: React Hooks + SWR
- **表单处理**: React Hook Form + Zod验证
- **动画**: Framer Motion + Tailwind Animation
- **Markdown**: MDX + @uiw/react-md-editor

#### 后端技术
- **数据库**: Supabase (PostgreSQL) with Row Level Security
- **认证**: NextAuth.js 5.0
  - 支持提供商: Google、GitHub、Apple、Email
- **支付集成**:
  - Creem (主要)
  - Stripe (国际)
  - Payssion V2 (俄罗斯市场)
- **视频生成提供商**:
  - Volcano Engine (Seedance模型)
  - APICore (Veo3模型)
  - fal.ai
  - Kie.ai
  - Ali Video
- **文件存储**: 
  - Supabase Storage
  - AWS S3 / Cloudflare R2
- **AI集成**:
  - Claude (Anthropic)
  - OpenAI
  - Azure OpenAI
  - DeepSeek

#### 部署和基础设施
- **部署平台**: Cloudflare Pages (standalone build)
- **边缘函数**: Cloudflare Workers
- **分析工具**:
  - Plausible Analytics
  - Google Analytics
  - Clarity
  - OpenPanel
- **监控**: 自定义事件追踪系统

## 四、代码规范

### 4.1 命名规范
- **文件命名**: 
  - 组件: PascalCase (如 `VideoGenerator.tsx`)
  - 工具函数: camelCase (如 `utils.ts`)
  - 路由页面: kebab-case (如 `image-to-video/`)
- **变量命名**:
  - 组件: PascalCase
  - 函数: camelCase
  - 常量: UPPER_SNAKE_CASE
  - CSS类: kebab-case

### 4.2 代码组织原则
- **分层架构**: Model-Service-Component 三层分离
- **单一职责**: 每个模块/组件只负责一个功能
- **DRY原则**: 避免重复代码，抽取公共逻辑
- **类型安全**: 充分利用TypeScript类型系统

### 4.3 开发规范
- **严格模式**: TypeScript strict mode启用
- **错误处理**: 统一的错误处理机制
- **日志记录**: 开发环境详细日志，生产环境精简日志
- **安全实践**: 
  - 不硬编码敏感信息
  - 输入验证（Zod schemas）
  - SQL注入防护（参数化查询）
  - XSS防护

### 4.4 Git工作流
- **分支管理**:
  - `main`: 主分支
  - `feature/*`: 功能分支
  - `fix/*`: 修复分支
- **提交规范**:
  - feat: 新功能
  - fix: 修复bug
  - docs: 文档更新
  - style: 代码格式
  - refactor: 重构
  - test: 测试
  - chore: 构建/工具

## 五、开发环境设置

### 5.1 环境要求
- Node.js 18+
- pnpm 8+
- PostgreSQL (通过Supabase)

### 5.2 环境变量
关键环境变量包括：
- 数据库连接 (SUPABASE_URL, SUPABASE_ANON_KEY)
- 认证密钥 (NEXTAUTH_SECRET, OAuth提供商密钥)
- 支付密钥 (STRIPE_SECRET_KEY, CREEM_API_KEY等)
- 视频生成API密钥 (各提供商API密钥)

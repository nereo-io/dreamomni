# 年卡月度使用限制方案设计

## 一、方案概述

### 目标
- 年卡用户一次性获得全年积分（保持现有逻辑）
- 增加每月使用上限限制
- 当月未使用额度不累积到下月
- 每月1日自动重置使用额度

### 核心原则
- 最小化改动现有代码
- 不影响现有积分发放逻辑
- 仅在积分扣减时增加限制检查

## 二、技术实现方案

### 1. 数据库改动

#### 新增表：monthly_credit_usage（月度积分使用记录）
```sql
CREATE TABLE monthly_credit_usage (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  user_uuid UUID NOT NULL REFERENCES users(uuid),
  month VARCHAR(7) NOT NULL, -- 格式：'2025-01'
  used_credits INTEGER NOT NULL DEFAULT 0,
  max_credits INTEGER NOT NULL, -- 月度限额
  plan_type VARCHAR(20) NOT NULL, -- 'yearly_mini' 或 'yearly_standard'
  created_at TIMESTAMP WITH TIME ZONE DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT CURRENT_TIMESTAMP,
  UNIQUE(user_uuid, month)
);

-- 添加索引提高查询性能
CREATE INDEX idx_monthly_credit_usage_user_month ON monthly_credit_usage(user_uuid, month);
```

#### 3.2 新增模型 `/models/monthlyUsage.ts`
```typescript
import { MonthlyUsage } from "@/types/monthlyUsage";
import { getSupabaseClient } from "./db";

// 获取或创建当月使用记录
export async function getOrCreateMonthlyUsage(
  userUuid: string,
  planType: 'yearly_mini' | 'yearly_standard'
): Promise<MonthlyUsage> {
  const supabase = getSupabaseClient();
  const currentMonth = new Date().toISOString().slice(0, 7); // '2025-01'
  
  // 根据计划类型设置月度限额
  const maxCredits = planType === 'yearly_mini' ? 400 : 2000;
  
  // 尝试获取现有记录
  const { data: existing, error: fetchError } = await supabase
    .from("monthly_credit_usage")
    .select("*")
    .eq("user_uuid", userUuid)
    .eq("month", currentMonth)
    .single();
  
  if (existing) {
    return existing;
  }
  
  // 创建新记录
  const { data: newRecord, error: insertError } = await supabase
    .from("monthly_credit_usage")
    .insert({
      user_uuid: userUuid,
      month: currentMonth,
      used_credits: 0,
      max_credits: maxCredits,
      plan_type: planType
    })
    .select()
    .single();
  
  if (insertError) {
    throw insertError;
  }
  
  return newRecord;
}

// 增加已使用积分
export async function increaseMonthlyUsage(
  userUuid: string,
  credits: number
): Promise<void> {
  const supabase = getSupabaseClient();
  const currentMonth = new Date().toISOString().slice(0, 7);
  
  const { error } = await supabase
    .from("monthly_credit_usage")
    .update({
      used_credits: supabase.sql`used_credits + ${credits}`,
      updated_at: new Date().toISOString()
    })
    .eq("user_uuid", userUuid)
    .eq("month", currentMonth);
  
  if (error) {
    throw error;
  }
}

// 检查是否超过月度限额
export async function checkMonthlyLimit(
  userUuid: string,
  requestedCredits: number,
  planType: 'yearly_mini' | 'yearly_standard'
): Promise<{ allowed: boolean; remaining: number; message?: string }> {
  const monthlyUsage = await getOrCreateMonthlyUsage(userUuid, planType);
  const remaining = monthlyUsage.max_credits - monthlyUsage.used_credits;
  
  if (requestedCredits > remaining) {
    return {
      allowed: false,
      remaining,
      message: `Monthly credit limit exceeded. You have ${remaining} credits remaining this month.`
    };
  }
  
  return {
    allowed: true,
    remaining: remaining - requestedCredits
  };
}
```

#### 3.3 修改 `/services/credit.ts`
在 `decreaseCredits` 函数中增加月度限制检查：

```typescript
export async function decreaseCredits({
  user_uuid,
  trans_type,
  credits,
}: {
  user_uuid: string;
  trans_type: CreditsTransType;
  credits: number;
}) {
  try {
    // 检查用户是否是年卡会员
    const membership = await findActiveMembershipByUserUuid(user_uuid);
    if (membership && membership.plan_type === 'yearly') {
      // 获取用户的订阅信息以确定具体的年卡类型
      const user = await findUserByUuid(user_uuid);
      const activeSubscription = await findActiveSubscriptionsByUserUuid(user_uuid);
      
      // 判断是 mini 还是 standard 年卡
      let yearlyPlanType: 'yearly_mini' | 'yearly_standard' = 'yearly_mini';
      if (activeSubscription && activeSubscription.length > 0) {
        const subscription = activeSubscription[0];
        if (subscription.product_name?.includes('Standard')) {
          yearlyPlanType = 'yearly_standard';
        }
      }
      
      // 检查月度限制
      const monthlyLimit = await checkMonthlyLimit(user_uuid, credits, yearlyPlanType);
      if (!monthlyLimit.allowed) {
        throw new Error(monthlyLimit.message || "Monthly credit limit exceeded");
      }
    }
    
    // 原有的积分扣减逻辑保持不变
    let order_no = "";
    let expired_at: string | undefined = undefined;
    let left_credits = 0;

    const userCredits = await getUserValidCredits(user_uuid);
    if (userCredits) {
      for (let i = 0, l = userCredits.length; i < l; i++) {
        const credit = userCredits[i];
        left_credits += credit.credits;

        if (left_credits >= credits) {
          order_no = credit.order_no;
          expired_at = credit.expired_at || undefined;
          break;
        }
      }
      if (left_credits < credits) {
        throw new Error("not enough credits");
      }
    }

    const new_credit: Credit = {
      trans_no: getSnowId(),
      created_at: getIsoTimestr(),
      user_uuid: user_uuid,
      trans_type: trans_type,
      credits: 0 - credits,
      order_no: order_no,
      expired_at: expired_at,
    };
    await insertCredit(new_credit);
    
    // 如果是年卡会员，更新月度使用记录
    if (membership && membership.plan_type === 'yearly') {
      await increaseMonthlyUsage(user_uuid, credits);
    }
  } catch (e) {
    console.log("decrease credits failed: ", e);
    throw e;
  }
}
```

## 五、风险评估

### 技术风险
1. **数据库性能**：新增查询可能影响性能
   - 缓解措施：添加适当索引，考虑缓存

2. **并发问题**：多个请求同时扣减积分
   - 缓解措施：使用数据库事务保证一致性

### 业务风险
1. **用户体验**：突然限制可能引起用户不满
   - 缓解措施：提前通知，界面显示剩余额度

2. **客服压力**：用户咨询增加
   - 缓解措施：准备FAQ，优化错误提示

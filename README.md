# GeminiOmni.tv

GeminiOmni.tv 是一个基于 Next.js 14 构建的 AI 媒体生成平台,围绕旗舰 **Gemini Omni** 视频模型,支持视频、图像、音乐与特效生成,并提供多模态创作界面 **Omni Studio**。

核心能力:
- 文本 / 图像转视频(Text / Image to Video)
- 文本 / 图像转图像(Text / Image to Image)
- 文本转音乐(Text to Music)
- 视频 / 图像特效(Video / Image Effects)
- Omni Studio 多模态合成(文本 + 图像 / 视频 / 音频 / 角色参考)

## 本地开发
- `pnpm dev`(默认端口 3000)

## 生产构建 / 本地预览
- `pnpm build`
- `pnpm start`

> 架构、命令与开发约定见 `CLAUDE.md`、`AGENTS.md` 与 `docs/DEVELOPMENT.md`。
